# Bharath Canteen - Next.js Application

Welcome to the Bharath Canteen web application! This project was migrated from a Google Stitch UI design and converted into a fully functional Next.js full-stack application.

## Overview

Bharath Canteen aims to bridge the gap between prestigious academic heritage and the fast-paced, vibrant energy of a modern culinary destination. The design targets tech-savvy students and faculty who demand efficiency without sacrificing the aesthetic quality of a premium food delivery service.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Fonts**: Plus Jakarta Sans (Headings) & Work Sans (Body)

## Design System
The application strictly adheres to the "Corporate Modern" design system defined in the Stitch export:
- **Primary Color**: `#B31217` (Deep Red - stimulates appetite and maintains brand alignment)
- **Secondary Color**: `#F2B974` (Gold - high-visibility highlights)
- **Background**: `#F5F5DC` (Cream - warm cafe-like atmosphere)
- **Typography**: Dual-font strategy for modern hierarchy and maximum readability.

## Getting Started

1. Make sure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Project Structure
- `src/app/` - Next.js App Router containing all pages (Home, Login, Admin, Menu, Cart, Orders) and backend API routes.
- `src/app/globals.css` - Global CSS including custom Tailwind theme variables from the design system.
